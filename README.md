# ATELIER — AI Personal Stylist 🤵

> Your AI stylist that dresses you for any occasion in seconds.

**Hackathon Project by Coco Team**

## What it does

- 📸 **Upload a photo** → AI identifies your outfit
- 🎭 **Say an occasion** → AI swaps your clothes instantly  
- 💑 **Describe your date** → AI recommends the perfect look
- 🛒 **Get Amazon links** → buy recommended items instantly
- 🔗 **Scan QR code** → share your AI-generated looks

## Tech Stack

- Frontend: Next.js 14 + Tailwind + shadcn/ui
- Backend: Supabase (DB + Storage + Auth)
- AI: GPT-4o Vision + Gemini 3 Pro Image
- Weather: OpenWeatherMap API
