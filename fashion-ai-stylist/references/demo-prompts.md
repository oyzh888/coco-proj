# COCO Demo Prompts — Copy-Paste Ready

## Demo A — Outfit Recognition + Shopping

```
Identify every clothing item I'm wearing. For each piece, tell me the style, color, and find me a similar item on Amazon.
```

## Demo B — Real-time Outfit Swap

```
I'm going to the opera tonight. Change my outfit to something elegant and sophisticated. Keep my face, hair, and pose exactly the same.
```

```
I'm heading to a beach party this weekend. Give me a summer look with swim shorts. Keep everything else the same.
```

```
I have a job interview at a tech startup tomorrow morning. Style me in something smart casual and confident.
```

## Demo C — Date-Aware Styling (Best Crowd Reaction)

```
I have a date tonight. He's a frat boy, athletic and casual. What should I wear? Show me on my photo.
```

```
I'm meeting someone for a first date. She's a sophisticated older woman who loves fine dining. Style me accordingly. Keep my face and pose the same.
```

```
I have a date with a creative art student. She's into indie fashion and vintage. Style me to match her vibe.
```

## Demo D — Outfit Combination Grid

```
This is my grey sweatshirt. Show me 4 different outfit combinations I can build around it for: casual, smart casual, date night, and outdoor.
```

```
I have this one piece of clothing. Give me 4 complete looks I can build around it for different occasions.
```

## Demo E — Style Analysis

```
Analyze my outfit. What's my style DNA? What does it say about me? What occasions is this perfect for, and what should I avoid?
```

## Demo F — Seasonal / Weather

```
I'm in San Francisco tomorrow. It'll be 58°F and foggy. Recommend a stylish but warm outfit and show it on me.
```

## Recommended 3-Minute Demo Flow

1. (30s) Send someone's photo → Demo A → AI identifies outfit + Amazon links
2. (45s) Same photo → Demo B opera prompt → outfit swap wow moment
3. (45s) Same photo → Demo C frat boy prompt → date styling + laughs
4. (30s) Point to Amazon links → "you can buy these right now"
5. (30s) QR code → "scan this to see everyone's looks from today"
